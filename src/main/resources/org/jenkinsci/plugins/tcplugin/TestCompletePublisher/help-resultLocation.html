The worksapace-based location of result file(s) to parse to JUnit format.
Depends on LogFormat you use in Log.SaveResultsAs method:
  • LogFormat value is 1: Unpacked storage. Set the path to directory with index.htm, root.xml and other result files
    Example: myTest/myTestResult
  • LogFormat value is 2: Multipart hypertext storage. Set the Location of result .mht file
    Example: myTest/myTestResult/result.mht

For more information see Exporting Test Results article.
