This step will run TestComplete with a batch script. See TestComplete Command Line guide here.
In general syntax is the following: 

  
  TestComplete.exe | TestExecute.exe [file_name [/run 
    [(/project:project_name) | (/project:project_name /projectitem:item_name) |
     (/project:project_name /test:test_name) |
     (/project:project_name /unit:unit_name /routine:routine_name)] 
    [/ExportLog:File_name] [/exit]]
    [/SilentMode [/ErrorLog:File_name] [/ForceConversion]] ]
    [Timeout:Time_in_seconds] [/ns] [/agpatcheroff]

You should provide
  • Path to TestComplete.exe or TestExecute.exe
  • Project or project suit to load
  • Space separated list of any other parameters

After finished click Generate Script button and verify result batch script. Visit this page for script example.
