This step will publish TestComplete results from the worksapace-based location of result file(s).

For more information see Exporting Test Results article.
Scenario is the following:
  • If .mht location given: decompress result .mht file to %WORKSPACE%\MHTParse folder
  • Generate JUnit test result report %WORKSPACE%\junitresult.xml
  • Publish junitresult.xml with 'Publish JUnit test result report' step
  • Optional:
    • Publish HTML result with 'Publish HTML reports' step
    • Publish screenshots with 'Publish HTML reports' step
    • If .mht location given: publish .mht file itself with 'Archive the artifacts' step
